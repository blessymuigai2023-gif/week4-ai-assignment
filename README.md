## what is AI
Artificial Intelligence (AI) is the simulation of human intelligence in machines that are programmed to think and learn.

## what is machine learning
Machine learning is a subset of AI that allows computers to learn from data without being explicitly programmed.

## some examples of AI applications
Examples of AI applications include virtual assistants, recommendation systems, and self-driving cars.
